About the game

This game was created as part of the Hackathon project at Interface 3 . The idea was to make a game that would bring coding closer to the people unfamiliar with the code.

The interns designed 3D robotic arm and a game in the Unity program and integrated it into a website.

The robotic arm moves the balls that contain the names of the programming languages. It is necessary to collect the balls and arrange them according to a certain rules.

How to play

The goal is to succeed in making the required combination in the right way.

Click on the piece of code you want for the mechanical arm to take it.

Make the right combinations to control the mechanical arm using its symbols, so as to put the piece of code back in the right place and with the good mechanical combination.
